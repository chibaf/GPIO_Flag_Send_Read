# GPIO_Flag_Send_Read
sending GPIO flags to another RasPi via socket

(RasPi1->RasPi2)

GPIO_flag_read_class.py,read_main.py: on RasPi2

GPIO_flag_send_class.py,send_main.py: on RasPi1

usage:

python3 read_main.py (on RasPi2)

python3 send_main.py (on RasPi1)
